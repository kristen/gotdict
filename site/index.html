GOTDict

  • GitHub repository
  • MOBI Download
  • Online Dictionary Index

A community built Kindle dictionary for George R. R. Martin's A Song of Ice and Fire series based on the work of KEC Krew. While a number of characters have X-Ray entries this dictionary aims to be much more extensive having information on houses, locations, ships, maps and much more. Currently there are {{ site.definitions.size }} entries in the dictionary.

The dictionary aims to the spoiler safe. Plot details have been removed from entries, characters who later die have past tense verbs changed to present. As this diminishes the usefulness of the dictionary further into the series it is hoped to create dictionary builds for each book, adding additional details as the story progresses.

Installation

Download the latest gotdict.mobi file and transfer to your device's /documents/ folder. Alternatively visit this page on your device and tap the download link.

While reading ASOIF books select "GOTDict" as your default dictionary (Settings > Device Options > Language and Dictionaries > Dictionaries)

Separate builds of the dictionary may be required for non-English devices, please file an issue if you have trouble.

Contributing

This project needs help to improve the quality and quantity of the entries. Use the online browser to read the dictionary's definitions, then fork me on GitHub and propose changes. Guidelines for entries will be drafted soon.
